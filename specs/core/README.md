# Core Index

## Purpose

`Core Index` is the entry point for runtime-kernel component documentation in `jido_code_ui`.

Its purpose is to:

- provide a single navigation surface for application bootstrap and supervision boundaries
- keep core ownership boundaries discoverable for implementation and review
- connect adjacent runtime component indexes from one place
